Tutorial

  1. En primer lugar, un endpoint es una URL específica a la que una aplicación puede enviar solicitudes para obtener datos o realizar acciones. En este caso, usaremos el endpoint de la API de la NASA para obtener la imagen del día: "https://api.nasa.gov/planetary/apod?api_key=TU_API_KEY"
  2. Luego solicitaremos una API KEY de la Nasa, la cual es gratis y no requiere realmente de un registro que podría comprometer tus datos
  3. Ahora hay que programarlo en JS. Para ello cree una variable Key en donde guarde nuestra clave que nos proporciono la NASA
  4. Para hacer la petición a la API tendrás que realizar un fetch, en donde debes cambiar la parte que dice "TU_API_KEY" por tu variable Key. Aquí hay dos cosas importantes que mencionar
    • fetch(url): Esta función realiza una solicitud HTTP a la API de la NASA y devuelve una promesa, que representa una respuesta futura.
    • await fetch(url): await se usa para esperar a que la promesa se resuelva y obtener la respuesta completa
  5. Finalmente, convertimos los datos recibidos de la Nasa a un objeto tipo JSON y los usamos
