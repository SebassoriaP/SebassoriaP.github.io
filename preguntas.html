HTML

Introduction

What is HTML?

HTML es un lenguaje de diseño que se utiliza principalmente para estructurar el diseño de una página web. Es decir es una herramienta fron-end que nos permite estructurar un sitio web.

Explain the head and the body tags.

La etiqueta "head" nos sirve para describir la meta información que tendrá nuestra página web; esto podría ser como el nombre del sitio web o información para ayudar al motor de búsqueda. En cuanto a la etiqueta "body" nos sirve para describir el contenido de nuestra página web; es decir, todo lo que se mostrará en la página web; aqui podemos encontrar las etiquetas "a","h1","p", entre otras.


Basic Tags

Explain the section, article, aside, header, footer tag?

Section: Es una etiqueta que se utiliza para dividir el contenido de una página web en diferentes secciones. Cada sección puede tener su propio header, footer, etc.

Article:Es usado principalmente para crear un blog, un post, una noticia, seccion de comentarios, etc. Al igual que la sección también puede contener header,p,footer,etc.

Aside: Contiene información relacionada con el contenido principal, como anuncios, links, etc.

Header:Es usado para definir la seccion inicial de una página, usualmente contiene el logo, el menú de navegación,etc

Footer: Se usa en la parte inferior de la página y usualmente contiene información de los contactos, copyright, etc

CSS

Introduction

What is CSS?

CSS al igual que HTML, es un lenguaje de fron-end que se especializa principalmente en darle estilo a nuestra página web.

What is the purpose of CSS?

Nos ayuda a proporcionar ciertas caracteristicas para nuestro sitio, como en el color de fondo, el tipo de letra, el tamaño, la posicion de los objetos, etc.

How do you apply a CSS to an HTML (explain the 3 options)

Para aplicar css, se puede:

  • Crear un archivo .css y vincularlo al archivo html
  • Dentro del HTML, en cada linea que queramos usar CSS llamamos al atributo style: "p style="color: blue; font-size: 20px;">Texto Azul."
  • Colocar una etiqueta "style", dentro del archivo html, lo cual nos permite utilizar css

What is a selector?

Un selector es un patrón de elementos que nos permite seleccionar un determinado tipo de elementos en el archivo, para poderles hacer modificaciones de estilo específicas para esee grupo

What is a CSS rule?

Una regla CSS es un conjunto de instrucciones que especifican como se debe de estilizar un específico grupo de elementos, Este esta conformado por dos partes, el selector(determina los elementos específicos) y el bloque de declaracion o propiedades (se ponen las propiedades de estilo que se desean)

Basic CSS

Explain a:

  • Tag Selector: Es un seleccionador que "selecciona" elementos según su etiqueta, ya sea "h1","h2","p", etc
  • Class Selector: Slecciona los elementos que pertenecen a una clase en especifico, su sintaxis es ".nombre_de_la_clase"
  • Tag and Class Selector: Es un tipo de selector que combina los dos anteriores, es decir que selecciona elementos de cierto tipo que pertenecen a cierta clase
  • Id Selector: Es un tipo de selector que selecciona elementos con un atributo de id, lo que significa que cierto elemento tiene un id y ciertos estilos elegidos para ese id

What is a class? What is an id? How are they related to CSS?

    Una clase es un identificador reusable que puede ser asignado a múltiples elementos, es útil a la hora de aplicar estilos a diferentes elementos. Un id es un identificador único que usualmente se le asigna a un solo elemento en la página Estos dos son relacionados a CSS ya que nos permiten aplicar estilos a ciertos elementos de una página web

Explain the CSS Box Model? Show an example.

Es un módulo que nos permite determinar cóo los elementos de una página web están estructurados, incluyendo el área de contenido, rellenos, los bordes y los márgenes. Estas modificaciones se pueden poner desde la etiqueta "style", en el "head"
